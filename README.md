# Python Code Project

## 项目简介
这是一个用于聊天模型的 Python 项目。
python3.11
## 安装步骤
1. 克隆这个仓库：
   ```bash
   git clone <repository-url>
   ```
2. 进入项目目录：
   ```bash
   cd python-code
   ```
3. 创建并激活虚拟环境：
   ```bash
   python -m venv aivenv
   aivenv\Scripts\activate
   ```
4. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法
在命令行中运行相应的 Python 文件，例如：
```bash
python 260125ty/chatPrompt.py
```
